# WinTop-AutoHotKey
Wintop is a simple autohotkey program that you can use to easily set any window always on top and/or click thru


#How To Use
  - While the window you would like to modify is the currently active window:
    - Alt+UpArrow = Make Window Always On Top
    - Alt+DownArrow = Make window AlwaysOnTop & Click Thru - applies transparency
    - Alt+(Left|Right)Arrow = Show WinTop 'manager' window - right click to clear settings on window
